## 📋 نظرة عامة

**GasBooking** هي منصة ويب حديثة ومبتكرة تهدف إلى تبسيط عملية حجز أسطوانات الغاز للمستخدمين. تم تصميم هذه المنصة لتوفير تجربة مستخدم سلسة ومريحة، مع واجهة عربية أنيقة ومتجاوبة تدعم جميع الأجهزة.

### 🎯 الهدف من المشروع

تم تطوير هذا المشروع كنموذج أولي (Prototype) لإثبات المفهوم وعرض إمكانيات التصميم والتفاعل. يهدف المشروع إلى:

- **عرض تجربة المستخدم المثلى** لحجز أسطوانات الغاز
- **توفير نموذج قابل للتطوير** للشركات والمطورين
- **إثبات جدوى الحلول الرقمية** في قطاع توزيع الغاز
- **تقديم أساس تقني قوي** للتطوير المستقبلي

## ⚠️ تنبيه مهم - طبيعة المشروع

> **هذا المشروع هو واجهة أمامية فقط (Frontend Only)**

### 🔍 ما يتضمنه المشروع حالياً:
- ✅ واجهة مستخدم تفاعلية كاملة
- ✅ تصميم متجاوب لجميع الأجهزة
- ✅ تخزين البيانات محلياً (Local Storage)
- ✅ محاكاة تدفق العمليات والحجوزات
- ✅ دعم الوضع الليلي والنهاري

### ❌ ما لا يتضمنه المشروع:
- ❌ خادم خلفي (Backend Server)
- ❌ قاعدة بيانات حقيقية
- ❌ نظام دفع فعلي
- ❌ تكامل مع خدمات خارجية
- ❌ نظام مصادقة حقيقي

### 💡 الغرض من التخزين المحلي:
يستخدم المشروع `localStorage` لمحاكاة عمليات حفظ واسترجاع البيانات، مما يتيح للمستخدمين تجربة المنصة بشكل كامل دون الحاجة لخادم خلفي. هذا يجعل المشروع مثالياً كنموذج أولي أو كأساس للتطوير.

## 🚀 البدء السريع

### 📋 المتطلبات الأساسية

قبل البدء، تأكد من توفر الأدوات التالية على نظامك:

- **Node.js** (الإصدار 16.0 أو أحدث)
- **npm** أو **yarn** أو **pnpm** (مدير الحزم)
- **Git** (لاستنساخ المشروع)

### 📥 تحميل المشروع

```bash
# استنساخ المشروع
git clone https://github.com/HadaniOmar/gas-booking.git

# الانتقال إلى مجلد المشروع
cd gas-booking

# تثبيت التبعيات (اختر أحد الخيارات)
npm install
# أو
yarn install
# أو
pnpm install
```

### 🏃‍♂️ تشغيل المشروع محلياً

```bash
# تشغيل خادم التطوير
npm run dev
# أو
yarn dev
# أو
pnpm dev
```

بعد تشغيل الأمر، ستجد المشروع متاحاً على: `http://localhost:5173`

### 🏗️ بناء المشروع للإنتاج

```bash
# بناء المشروع
npm run build
# أو
yarn build
# أو
pnpm build
```

ستجد الملفات المبنية في مجلد `dist/`
